# 回溯算法
    本质：是暴力穷举，如何没有遗漏的穷举完所有可能是回溯算法的关键
    实现：用一个决策树来表示穷举的过程：
    每个节点表示一个决策，需要维护走到当前节点的时候已经走了【那些路径】，已经还可以【选择】的列表。

伪码
```
backtrack函数：可以理解为在这个决策树节点上游走的指针。

void backtrack(已走路径,可选列表){
    //判断是否满足终止条件
    if(满足终止条件){
        // 满足终止条件，则输出结果
    }
    for(选择：选择列表){
        // 1.做选择。
        // 2.递归
        backtrack(选择后的已走路径, 选择列表);
        // 3.撤销选择
    }
}
```
