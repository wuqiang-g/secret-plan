# 最小生成树
## 1.生成树
    概念：在一个无向连通图中，一颗生成树有以下概念
    1. 包含图的所有节点。
    2. 边的数量是节点数量-1。
    3. 连通且无环。
## 2.最小生成树
    图是加权（无向连通）图。
    1. 最小生成树：生成树中边权之和最小的树

## 3.Kruskal算法
    （贪心思想）先对图中所有边进行排序，然后借助UF并查集算法构造

前置知识：有一组点和一组边组成的图，如何判断这个图是否是树？
    
    1.图不成环，且只有一个连通分量，则代表这个图是树。
    做法：用UF初始化一个并查集，连通边的两个点之前，先判断这两个点是否连通，
    如果已连通再去连接这条边，则一定成环。最后判断并查集中连通分量是否为1

Kruskal算法在【以图判树】的基础上，先对边权重从低到高排序，然后依次连接边（成环的边跳过），最后看并查集中连通分量是否为1，为1则返回收集到的边最小边权重和。
