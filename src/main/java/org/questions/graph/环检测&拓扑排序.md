## DFS 环检测

    onPath数组记录遍历过的节点，再次遍历到该节点，说明存在环

例题：

[207. 课程表](https://leetcode.cn/problems/course-schedule/description/) DFS遍历，借助onPath数组判断成环，借助visited数据解决冗余访问节点。

```java
class Solution {
    boolean hasCycle = false;
    boolean[] onPath ;
    boolean[] visited ;

    public boolean canFinish(int numCourses, int[][] prerequisites) {
        onPath = new boolean[numCourses];
        visited = new boolean[numCourses];
        //有向图的环检测算法，numCourses是节点，prerequisites是有向边

        List<Integer>[] graph = buildGraph(numCourses,prerequisites);

        for(int i = 0 ; i < numCourses ;i ++){
            traverse(graph,i);
        }
        //不存在环，则能完成学业
        return !hasCycle;
    }

    void traverse(List<Integer>[] graph,int s){
        if(hasCycle){
            return;
        }
        if(onPath[s]){
            hasCycle = true;
            return;
        }
        if(visited[s]){
            return;
        }
        //前序位置
        onPath[s] = true;
        visited[s] = true;
        for(int e : graph[s]){
            traverse(graph,e);
        }
        //后序位置
        onPath[s] = false;

    }

    //邻接表构建图
    List<Integer>[] buildGraph(int numCourses, int[][] prerequisites){
        List<Integer>[] graph = new LinkedList[numCourses];
        for(int i = 0 ; i < numCourses; i++){
            graph[i] = new LinkedList<Integer>();
        }
        for(int[] edge : prerequisites){
            int form = edge[1];
            int to = edge[0];
            graph[form].add(to);
        }
        return graph;
    }
}
```

## 拓扑排序 DFS
    进行拓扑排序之前，先要确保图中没有环。无环才能拓扑排序
    图的「逆后序遍历」顺序，就是拓扑排序的结果

例题：

[210. 课程表 II](https://leetcode.cn/problems/course-schedule-ii/description/) 先成环判断，在后序位置收集接节点，再逆序输出就是拓扑排序
```java
class Solution {
        // 记录后序遍历结果
    List<Integer> postorder = new ArrayList<>();
    // 记录是否存在环
    boolean hasCycle = false;
    boolean[] visited, onPath;
    public int[] findOrder(int numCourses, int[][] prerequisites) {
        List<Integer>[] graph = buildGraph(numCourses, prerequisites);
        visited = new boolean[numCourses];
        onPath = new boolean[numCourses];
        // 遍历图
        for (int i = 0; i < numCourses; i++) {
            traverse(graph, i);
        }
        // 有环图无法进行拓扑排序
        if (hasCycle) {
            return new int[]{};
        }
        //无环图，后序遍历结果，反转，则是拓扑排序
        Collections.reverse(postorder);
        int[] res = new int[numCourses];
        for(int i = 0 ; i < postorder.size() ; i++){
            res[i] = postorder.get(i);
        }
        return res;
    }

        // 图遍历函数
    void traverse(List<Integer>[] graph, int s) {
        if (onPath[s]) {
            // 发现环
            hasCycle = true;
        }
        if (visited[s] || hasCycle) {
            return;
        }
        // 前序遍历位置
        onPath[s] = true;
        visited[s] = true;
        for (int t : graph[s]) {
            traverse(graph, t);
        }
        // 后序遍历位置
        postorder.add(s);
        onPath[s] = false;
    }

    List<Integer>[] buildGraph(int numCourses, int[][] prerequisites) {
        // 图中共有 numCourses 个节点
        List<Integer>[] graph = new LinkedList[numCourses];
        for (int i = 0; i < numCourses; i++) {
            graph[i] = new LinkedList<>();
        }
        for (int[] edge : prerequisites) {
            int from = edge[1], to = edge[0];
            // 添加一条从 from 指向 to 的有向边
            // 边的方向是「被依赖」关系，即修完课程 from 才能修课程 to
            graph[from].add(to);
        }
        return graph;
    }
}
```

## 环检测 BFS
    借助【入度：指向该节点的其他节点】indegree数组来遍历图，入度为0则加入队列，大于0的话出队后，入度减一。
    如果出队的节点数等于图节点数，则不存在环。

例题：

[207. 课程表](https://leetcode.cn/problems/course-schedule/description/) 借助indegree数组维护每个节点的入度数量。入度为0加入队列，依次出队，最后判断出队数和图节点数

```java
class Solution {
    //BFS方式
    int[] indegree;

    public boolean canFinish(int numCourses, int[][] prerequisites) {
        indegree = new int[numCourses];
        List<Integer>[] graph = buildGraph(numCourses,prerequisites);
        //构建所有节点的入度数量
        for(int i = 0 ; i < numCourses ; i++){
            for(int to : graph[i]){
                indegree[to]++;
            }
        }
        Queue<Integer> q = new LinkedList<>();
        for(int i = 0 ; i < numCourses ; i++){
            if(indegree[i] == 0){
                //所有入度为0的加入队列。
                q.offer(i);
            }
        }
        int count = 0;
        while(!q.isEmpty()){
            //出队。遍历的节点数+1
            int cur = q.poll();
            count++;
            for(int next : graph[cur]){
                //cur出队了，以cur作为前序节点的接的入度则减一
                indegree[next]--;
                //next没有入度了，则可以加入队列，一会出队遍历next
                if(indegree[next] == 0){
                    q.offer(next);
                }
            }
        }

        //不存在环，则能完成学业
        return count == numCourses;
    }

    //邻接表构建图
    List<Integer>[] buildGraph(int numCourses, int[][] prerequisites){
        List<Integer>[] graph = new LinkedList[numCourses];
        for(int i = 0 ; i < numCourses; i++){
            graph[i] = new LinkedList<Integer>();
        }
        for(int[] edge : prerequisites){
            int form = edge[1];
            int to = edge[0];
            graph[form].add(to);
        }
        return graph;
    }
}
```

## 拓扑排序 BSF
    BSF 的遍历顺序就是拓扑排序的结果，前提是不存在环

例题：

[210. 课程表 II](https://leetcode.cn/problems/course-schedule-ii/description/) 借助indegree数组维护每个节点的入度数量。入度为0加入队列，依次出队，最后判断出队数和图节点数。相等成环，在出队的地方收集拓扑排序结果
```java
class Solution {
    //BFS方式
    int[] indegree;

    public int[] findOrder(int numCourses, int[][] prerequisites) {
        indegree = new int[numCourses];
        List<Integer>[] graph = buildGraph(numCourses,prerequisites);
        //构建所有节点的入度数量
        for(int i = 0 ; i < numCourses ; i++){
            for(int to : graph[i]){
                indegree[to]++;
            }
        }
        Queue<Integer> q = new LinkedList<>();
        for(int i = 0 ; i < numCourses ; i++){
            if(indegree[i] == 0){
                //所有入度为0的加入队列。
                q.offer(i);
            }
        }
        int count = 0;
        int[] res = new int[numCourses];
        while(!q.isEmpty()){
            //出队。遍历的节点数+1
            int cur = q.poll();
            //收集拓扑排序结果
            res[count] = cur;
            count++;
            for(int next : graph[cur]){
                //cur出队了，以cur作为前序节点的接的入度则减一
                indegree[next]--;
                //next没有入度了，则可以加入队列，一会出队遍历next
                if(indegree[next] == 0){
                    q.offer(next);
                }
            }
        }
        if(count != numCourses){
            //存在环
            return new int[]{};
        }
        return res;
    }

    List<Integer>[] buildGraph(int numCourses, int[][] prerequisites) {
        // 图中共有 numCourses 个节点
        List<Integer>[] graph = new LinkedList[numCourses];
        for (int i = 0; i < numCourses; i++) {
            graph[i] = new LinkedList<>();
        }
        for (int[] edge : prerequisites) {
            int from = edge[1], to = edge[0];
            // 添加一条从 from 指向 to 的有向边
            // 边的方向是「被依赖」关系，即修完课程 from 才能修课程 to
            graph[from].add(to);
        }
        return graph;
    }
}
```
